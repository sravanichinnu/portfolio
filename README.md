# Sravani Namburu — Portfolio

Source for my personal portfolio site, built as a static `index.html`.

**Live site:** https://sravanichinnu.github.io/portfolio/

## Structure
- `index.html` — the site (single self-contained page)
- `.nojekyll` — tells GitHub Pages to serve the site as-is (skip Jekyll processing)
- `images/` — photo and project screenshots

## Updating
Edit `index.html` directly and push to `main`. GitHub Pages redeploys automatically within a minute or two.
